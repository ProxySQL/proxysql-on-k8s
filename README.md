# proxysql-on-k8s

Kubernetes operator and Helm charts for [ProxySQL 3.x](https://proxysql.com/)
(MySQL + PostgreSQL).

> **Scaffolding repository.** The initial implementation lands as a single
> pull request — see open PRs.
>
> For the legacy v1 demo charts, see
> [`ProxySQL/kubernetes`](https://github.com/ProxySQL/kubernetes).
